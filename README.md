# Fake News Detection using Deep Learning (LSTM)

## Project Overview
This project detects whether a news article is Fake or Real using a Long Short-Term Memory (LSTM) deep learning model.

## Features
- Data preprocessing
- Text cleaning
- Tokenization
- LSTM model
- Fake/Real prediction
- Streamlit web application

## Technologies Used
- Python
- TensorFlow
- Keras
- Pandas
- NumPy
- Scikit-learn
- NLTK
- Streamlit

## Model Accuracy
Test Accuracy: ~99%

## Dataset
Fake and Real News Dataset

## How to Run

Install dependencies:

pip install -r requirements.txt

Run:

streamlit run app.py
